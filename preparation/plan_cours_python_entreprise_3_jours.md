# Programme de formation Python debutant (Entreprise) - 3 jours (version allegee)

## Informations generales
- Duree : 3 jours (21h)
- Public : 6 participants debutants complets
- Objectif : automatiser des taches metier via scripts Python simples et fiables
- Format : 30% theorie / 70% pratique

## Perimetre retenu
- Inclus : automatisation fichiers, traitement CSV/Excel, initiation API guidee
- Exclu : data visualisation
- Contrainte : progression sans surcharge (socle solide)

## Objectifs pedagogiques finaux
A la fin des 3 jours, chaque participant doit savoir :
- automatiser un traitement simple sur fichiers CSV/Excel ;
- structurer son code avec fonctions ;
- produire un script robuste de base (erreurs + logs simples).

## Charge pedagogique maitrisee
- Maximum 2 notions nouvelles fortes par demi-journee
- 60% pratique guidee / 40% pratique autonome
- Quiz flash + correction collective a chaque demi-journee
- Livrable concret a produire chaque jour

## Deroule detaille

### Jour 1 - Fondations Python et premier script
**Contenu**
- Environnement Windows + VS Code + execution script
- Variables, conditions, boucles, listes/dictionnaires
- Structure minimale d'un script lisible

**Atelier metier**
- Automatisation simple d'une tache repetitive

**Livrable du jour**
- 1 script executable + mode d'emploi court (5 lignes)

---

### Jour 2 - Traitement de donnees CSV/Excel
**Contenu**
- Fonctions et modularisation
- Lecture/ecriture CSV
- Excel usage simple : chargement, nettoyage, export
- Controles de donnees de base

**Atelier metier**
- Transformer un export metier en sortie propre (nettoyage + KPI simples)

**Livrable du jour**
- 1 script de traitement reutilisable

---

### Jour 3 - API initiation + robustesse + mini-projet
**Contenu**
- API initiation guidee : GET simple + token + export CSV
- Pas de pagination avancee
- Robustesse : try/except, validations minimales, logging basique
- Parametrage simple du script

**Atelier fil rouge**
- Mini-projet a portee reduite :
  - source principale CSV/Excel
  - option API courte guidee
  - sortie rapport CSV/Excel

**Livrable final**
- `script_final.py`
- `README_execution.md`
- `donnees_exemple.csv`
- `resultat_exemple.csv`

## Contrat de sortie
- Plan officiel : version 3 jours allegee
- Livrables individuels obligatoires :
  - `script_final.py`
  - `README_execution.md`
  - `donnees_exemple.csv` / `resultat_exemple.csv`
- Evaluation finale : competences minimales atteintes / non atteintes

## Modes degrades prevus
- Installation bloquee : environnement pret a l'emploi
- API indisponible : JSON local simulant la reponse API
- Ecart de niveau : binomes pilot / co-pilot
- Retard planning : API reduite a demonstration guidee

## Validation finale
- Chaque demi-journee tient en 3h30 avec marge de 20 min
- Un debutant complet termine l'atelier du Jour 1
- Le livrable Jour 2 produit un fichier de sortie correct
- Le livrable Jour 3 s'execute de facon autonome
